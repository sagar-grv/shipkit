What's Inside

Everything a production app needs — pre-configured and ready to go.

🤖

AI Agent Protocol

Your AI agent reads AGENTS.md at session start — it knows your tech stack, pipeline rules, and project state. Works with Claude Code, Cursor, Copilot, and any AI coding agent.

🔄

CI/CD Pipeline

GitHub Actions workflow: lint → type-check → test → build → deploy. Automatically runs on every push and pull request. Pre-configured for Vercel, Netlify, Railway, and more.

🔒

Security by Default

CodeQL vulnerability scanning on every PR. Dependabot for weekly dependency updates. Pre-push security review gate. Built-in ethical use license terms.

🧩

Pre-Commit Hooks

Husky + lint-staged run linting and formatting before every commit. Catch issues before they reach CI. Zero config — works automatically.

📋

Session Continuity

Your AI agent tracks what was done, what's next, and what's broken. ROADMAP.md, BUGS.md, and LAST_SESSION.md keep everyone in sync.

🔍

Security Reviewer

A dedicated AI agent reviews every diff for secrets, SQL injection, RLS bypass, and OWASP Top 10 before you push. "review security" is one command.

Get Started in 2 Minutes

Three commands. Zero configuration. Works in any project directory.

Step 1
cd your-project && npx shipkit-pipe setup
Interactive wizard — ~5 questions
Step 2
git add -A && git commit -m "init shipkit"
Your pipeline, agent configs, and security checks are ready
Step 3
Open in your AI agent → say "plan: login page"
Your agent reads AGENTS.md and knows everything
