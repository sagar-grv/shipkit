Why doesn't it ask anything?

Because your project already tells ShipKit everything it needs to know.

If you have a lint script, CI gets a lint step. No test script? No test step. Your CI never fails because of missing commands.
pnpm-lock.yaml → pnpm. yarn.lock → yarn. bun.lockb → bun. No guessing.
Reads origin URL to get your GitHub owner/repo. Supports GitHub, GitLab, and Bitbucket.
Found vercel.json? Health checks ping your Vercel URL. Found fly.toml? It knows your Fly.io app name.
Run npx shipkit-pipe -i for interactive mode. But 95% of users never need it.
