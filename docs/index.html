AI Agent Protocol

Your Agent Remembers Everything

ShipKit generates AGENTS.md — a universal protocol your AI reads every session. It knows your stack, your pipeline rules, and exactly where you left off.

  • Works with Claude Code, Cursor, Copilot, Cline, OpenCode, Aider
  • Session continuity via LAST_SESSION.md + BUGS.md + ROADMAP.md
  • No context window wasted — agent reads only what matters
CI/CD Pipeline

Push to Ship. No Config.

Pre-built GitHub Actions workflows run linting, type-checking, tests, and build on every push. Add preview URLs, E2E tests, and deploy automatically.

  • Vercel, Netlify, Railway, Fly.io, Render, Cloudflare, AWS, GCP
  • Husky pre-commit hooks catch issues before CI
  • Playwright E2E testing workflow included
# .github/workflows/ci.yml
on: [push, pull_request]
jobs:
  ci:
    steps:
      - run: npm lint
      - run: npm test
      - run: npm run build
      - run: npx playwright test
      - run: npx tsc --noEmit

Security by Default

Built-In Security Reviewer

A dedicated AI agent reviews every diff before you push. Secrets, SQL injection, RLS policies, OWASP Top 10 — caught before they reach production.

  • CodeQL vulnerability scanning on every PR
  • Dependabot for weekly npm/actions updates
  • One command: "review security" before push
Async Monitoring

Your SRE Never Sleeps

Every session starts with a Monitor Agent check — it reads Sentry, Vercel logs, and GitHub Actions status. If something broke overnight, you know before you type a line.

  • Post-deploy canary checks every session
  • Root cause analysis on every error
  • Graceful degradation when APIs rate-limit

Quick Start

Ship in 3 Commands

Works in any project directory. No account needed. No SaaS sign-up.

1
npx shipkit-pipe setup
Interactive wizard detects your stack (~30 seconds, ~5 questions)
2
git add -A && git commit -m "init: shipkit"
Your pipeline, security scanning, agent prompts, and pre-commit hooks are ready
3
git push origin main
CI runs lint → type-check → test → build → deploy + CodeQL security scan

After setup, open your project in your AI agent and say "plan: login page" — your agent already knows everything.
