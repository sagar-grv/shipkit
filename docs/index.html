Why doesn't it ask anything?

Because your project already tells ShipKit everything it needs to know.

📋 package.json scripts

If you have a lint script, CI gets a lint step. No test script? No test step. Your CI never fails because of missing commands.

📂 Lock files

pnpm-lock.yaml → uses pnpm. yarn.lock → uses yarn. bun.lockb → uses bun. No guessing.

🔗 Git remote

Reads your origin URL to get your GitHub owner/repo. No need to type it.

🚀 Deploy config

Found vercel.json? Health checks will ping your Vercel URL. Found fly.toml? It knows your Fly.io app name.

💡 Want to customize?

What your automated team does

Six systems that run automatically after you push. You never check manually again.

⚡

Smart CI/CD

Generates GitHub Actions that match YOUR scripts. Only includes steps that exist. Lint script? Lint step. No lint script? No lint step. Your CI never fails because of bad config.

🏥

Health Monitoring

Pings your production site every 6 hours. If it's down → auto-creates a GitHub Issue. When it recovers → auto-closes the issue. You know before your users.

🔒

Security Scanning

CodeQL scans every push for vulnerabilities. Dependabot keeps deps updated with auto-PRs. No manual checking.

🤖

AI Agent Memory

Generates AGENTS.md and shipkit.json so your AI agent (Claude, Cursor, Copilot) knows your stack, your rules, and what happened last session.

📦

Auto-detect Everything

Framework, package manager, Node version, deploy platform, git remote, available scripts — all detected from your existing files. Zero config.

Verify Command

Three seconds. Four steps.

From install to working pipeline — no wizard, no prompts, no signup.

1️⃣ Run the command

npx shipkit-pipe — in any directory. No install needed. Runs via npx.

2️⃣ Auto-detect

Reads package.json, lock files, git remote, deploy config, .nvmrc. Detects your framework, package manager, node version, and which scripts you have.

3️⃣ Generate

Creates 5-7 files: CI workflows, health checks, security scanning, Dependabot config, and AI agent instructions. Only generates what's relevant.

4️⃣ Push and forget

git push — GitHub Actions takes over. CI on every PR. Health checks every 6h. Dependabot keeps deps fresh. CodeQL finds vulnerabilities.

Verify everything works
